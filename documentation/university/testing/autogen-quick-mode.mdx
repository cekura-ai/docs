## What You'll Learn

How to use Cekura's AI to automatically generate test scenarios from your agent description, saving hours of manual evaluator creation.

## Why Autogen?

Writing evaluators manually works for a few test cases, but to cover the breadth of conversations your agent handles, you need dozens or hundreds. Cekura can generate them for you by analyzing your agent's system prompt and understanding the workflows it supports.

## Generate Evaluators

<Steps>
  <Step title="Go to Evaluators Tab">
    Select your agent and navigate to the **Evaluators** tab.
  </Step>

  <Step title="Click Generate Evaluators">
    Click the **Generate Evaluators** button to open the generation dialog.
  </Step>

  <Step title="Configure Generation">
    - **Number of scenarios**: How many evaluators to generate (start with 10-20)
    - **Scenario type**: Choose between standard scenarios or red-teaming
    - **Modality**: Text (faster, cheaper) or Voice

    <Tip>
    A good agent description is the key to high-quality generated evaluators. Make sure you've pasted your full system prompt in the agent description.
    </Tip>
  </Step>

  <Step title="Generate and Review">
    Click **Generate**. Cekura analyzes your agent description and creates evaluators covering different conversation flows, edge cases, and user intents.

    Review each generated evaluator — edit instructions or expected outcomes if needed.
  </Step>
</Steps>

## Quick Mode: Text-Based Testing

Text mode lets you run evaluators via chat instead of voice calls. This is the fastest way to iterate on your test suite.

**Benefits:**
- **10x faster** than voice testing
- **90% cheaper** — no telephony costs
- **Same evaluation** — metrics and expected outcomes work identically

To use text mode, ensure your agent has a chat integration configured (provider chat or custom WebSocket). Then select **Text** as the modality when running evaluators.

## Dynamic Variables in Generated Evaluators

If your agent description contains `{{variableName}}` placeholders, Cekura automatically:

1. Detects the variables
2. Generates realistic values for each evaluator
3. Creates matching test profiles and mock tool data

This ensures generated evaluators are internally consistent — the evaluator provides the same identity data that your agent's tools will look up.
