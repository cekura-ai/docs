---
title: Update Metric (Partial)
description: "Partially update a metric. Includes agent-level config fields that write through to the associated agent."
openapi: patch /test_framework/v1/metrics/{id}/
---

Three fields write through to the metric's associated agent when provided:

- **`pronunciation_words`** — list of `[word, phonemes]` pairs for the pronunciation analysis metric (e.g. `[["HIPAA", "hɪpə"]]`).
- **`spelling_word_types`** — list of word-category strings for the spelling analysis metric (e.g. `["medical", "technical"]`).
- **`hallucination_kb_files`** — list of KnowledgeBaseFile IDs used by the hallucination detection metric. Replaces the current set on the agent.
