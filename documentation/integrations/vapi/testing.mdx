<Note>
  You can test your agents directly on our platform without configuring API keys. Cekura will use internally generated transcripts for testing purposes. For a simpler setup without API keys, check out our guide on [testing custom agents](/documentation/guides/testing-agents/overview).
</Note>

## Prerequisites
- A Cekura account
- VAPI API credentials

## Setting Up VAPI Integration

### 1. Accessing Organization Settings
1. Log in to your [Cekura Dashboard](https://dashboard.cekura.ai)
2. Click on the **Organization** button in the navigation menu
3. Navigate to the **Settings** tab

<img src="/images/org-button-click.png" alt="Organization Settings Navigation" />

### 2. Configuring VAPI Authentication
1. Locate the **VAPI Integration** section
2. Enter your VAPI authentication key in the designated field

<img src="/images/vapi/1.png" alt="VAPI API Configuration" />

## Testing Your Agent

### 1. Select VAPI as your provider when creating a new agent

1. Navigate to the **Agents** section
2. Click **Add Agent** button
3. Select **VAPI** as your provider
4. Copy your assistant id from VAPI into the Assistant ID field

<img src="/images/create-agent.png" alt="VAPI API Configuration" />
