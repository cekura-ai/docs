## What You'll Learn

How to use personalities to simulate diverse real-world callers and test your agent's resilience to challenging conditions.

## What is a Personality?

A personality defines how an evaluator behaves during a conversation. It controls:

- **Language** — which language the evaluator speaks
- **Background noise** — street noise, cafe ambiance, office sounds
- **Interruption patterns** — whether the evaluator interrupts mid-sentence
- **Speaking pace** — slow and deliberate to fast and hurried
- **Emotional tone** — calm, frustrated, or urgent

## Built-In Personalities

Cekura provides a library of pre-defined personalities:

| Personality | Characteristics | Tests For |
|------------|-----------------|-----------|
| Normal Male/Female | Calm, clear speech | Baseline performance |
| Interrupter | Urgent tone, interruptions | Interruption handling |
| Background Noise | Street/cafe ambient noise | Real-world audio quality |
| Slow Speaker | Slower pace, deliberate speech | Patience and timing |
| Spanglish | Mixed English/Spanish | Multilingual handling |

## Assigning Personalities

When creating or editing an evaluator, select a personality from the dropdown. Each evaluator gets one personality, ensuring consistent behavior across test runs.

## Recommended Testing Mix

For comprehensive coverage, distribute your evaluators across personality types:

| Percentage | Personality Type | Purpose |
|-----------|-----------------|---------|
| 60% | Standard, clear communication | Baseline and core flow testing |
| 20% | Challenging conditions (noise, interruptions) | Robustness testing |
| 10% | Non-native speakers or accent variations | Accessibility testing |
| 10% | Edge cases (frustrated, very fast/slow) | Stress testing |

## Custom Personalities

Need a personality that doesn't exist in the library? You can create custom personalities tailored to your specific user base.

See [Creating Custom Personalities](/documentation/advanced/creating-custom-personalities) for details.

<Tip>
When running evaluators, you can select multiple evaluators with different personalities at once. This gives you a comprehensive view of how your agent performs across caller types in a single run.
</Tip>
