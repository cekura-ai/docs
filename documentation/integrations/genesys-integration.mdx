## Overview

Connect your Genesys Cloud contact center platform to Cekura. Test your Genesys-based voice agents with automated call simulation and capture tool call data for evaluation.

## Prerequisites

- A Cekura account
- A Genesys Cloud account with API access
- Your Genesys Cloud **Client ID** and **Client Secret** (OAuth credentials)
- Your Genesys Cloud **Region** (e.g., `mypurecloud.com`)

## Supported Capabilities

| Capability | Supported |
|------------|-----------|
| Voice (Telephony) | Yes |
| Tool Calls | Yes |
| Text/Chat | No |
| WebRTC | No |

## Configuration

<Steps>
<Step title="Create a New Agent">
1. Navigate to the **Agents** section using the side navigation bar.
2. Click the **Add Agent** or **Create Agent** button.
</Step>

<Step title="Configure Genesys Credentials">
1. Navigate to the **Agent Settings** panel.
2. In the **Provider** dropdown, select **Genesys**.
3. Enter your Genesys Cloud credentials:
   - **Genesys Client Secret**: Your OAuth client secret
   - **Genesys Client ID**: Your OAuth client ID
   - **Genesys Region**: Your Genesys Cloud region (e.g., `mypurecloud.com`)
4. Click **Save** to configure the agent.

<Note>
All three fields (Client Secret, Client ID, and Region) are required to complete the Genesys integration.
</Note>
</Step>

<Step title="Run Tests">
Once your Genesys agent is configured:

1. Go to **Simulation > Evaluator**.
2. Click **Generate Evaluators** to auto-create basic metrics.
3. Choose an **Evaluator** from the list (or create one).
4. Click the **Play** button to start your first test.
5. Monitor the test in **Simulation > Runs Overview**.
6. View **transcripts, audio, and metrics** in **Results**.
</Step>
</Steps>

## Finding Your Genesys Credentials

To get your OAuth credentials from Genesys Cloud:

1. Log in to your Genesys Cloud Admin portal.
2. Navigate to **Admin > Integrations > OAuth**.
3. Create a new OAuth client or use an existing one with appropriate permissions.
4. Copy the **Client ID** and **Client Secret**.
5. Note your **Region** from your Genesys Cloud URL (e.g., `mypurecloud.com`, `mypurecloud.de`, `mypurecloud.jp`).

<Tip>
Make sure your OAuth client has the necessary permissions to access the Genesys APIs that Cekura will use for testing.
</Tip>
