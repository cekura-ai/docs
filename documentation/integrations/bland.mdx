## Overview

Connect Bland AI to Cekura to run voice and chat evaluations. The integration supports automatic agent configuration, prompt sync, outbound test calls, and mock tools.

<Note>
Use your Bland **Persona ID** for voice testing and your **Pathway ID** for chat testing.
</Note>

## Prerequisites

- A Bland AI API key
- A published Bland Persona and its Persona ID
- A Bland phone number for voice testing
- A published Bland Pathway and its Pathway ID for chat testing

## Configure Bland in Cekura

<Steps>
  <Step title="Create or open an agent">
    In Cekura, go to **Agents** and create a new agent or open an existing one.
  </Step>

  <Step title="Connect your Bland Persona">
    In **Agent Settings → Provider**, select **Bland AI**. Enter the Persona ID as the **External Assistant ID**, add your Bland API key, and save the agent.

    <img src="/images/bland/agent-settings.png" alt="Bland AI provider settings in Cekura" />
  </Step>
</Steps>

When provider configuration is enabled, Cekura can import the Persona's name, prompt, language, phone number, dynamic variables, knowledge bases, and supported tools. Enable **Auto-sync Prompt** to keep the Cekura agent description aligned with Bland.

## Voice testing and automatic outbound calls

To run voice evaluations, configure the Bland phone number under **Telephony Settings**.

For Cekura to initiate each Bland test call automatically:

1. Leave **Inbound** disabled.
2. Enable **Outbound Auto Call**.
3. Save the agent and run an evaluator using the **Telephony** connection.

<img src="/images/bland/auto-outbound-call.png" alt="Bland automatic outbound call settings in Cekura" />

Cekura sends configured dynamic variables with the call and records the transcript, audio, metrics, and tool calls in the result. See [Automatic Outbound Calls](/documentation/guides/testing-agents/outbound-auto-call) for the complete workflow.

## Chat testing

Bland chat testing uses a conversational Pathway instead of the voice Persona.

<Steps>
  <Step title="Add the Pathway ID">
    In **Agent Settings → Bland Chat Settings**, enter the Pathway ID and save the agent.
  </Step>

  <Step title="Run a chat evaluation">
    Open an evaluator, click **Run**, select **Bland** under **Chat**, and start the run.

    <img src="/images/bland/configure-run-chat.png" alt="Configure Run dialog with Bland chat selected" />
  </Step>
</Steps>

See [Chat Testing](/documentation/integrations/chat-testing#bland) for more details.

## Mock tools

Use **Auto-Fetch** in the agent's **Mock Tools** section to import supported Bland tools and generate mock responses. During an outbound test, Cekura uses temporary tool copies for that run, leaving the Persona's production tools unchanged.

Learn more in [Mock Tools](/documentation/key-concepts/evaluators/mock-tools).

<Info>
Bland does not currently support automatic import of production calls into Cekura.
</Info>
