This is for the **Claude app** at [claude.ai](https://claude.ai) — web, mobile, and the desktop chat. Add Cekura as a **custom connector** (MCP tools), and optionally install the **plugin** for Cekura Skills.

<Note>
For the full developer experience — slash commands, plugin hooks, and update support — use [Claude Code](/mcp/claude-code-guide) instead. This page is the chat-app path.
</Note>

## Add the Cekura connector

This gives Claude the Cekura MCP tools (list agents, create metrics, run tests, inspect results).

<Steps>
  <Step title="Open Connectors">
    In the Claude app, go to **Customize → Connectors**.
  </Step>

  <Step title="Add a custom connector">
    Click **+ → Add custom connector**, and set the URL to `https://api.cekura.ai/mcp`.
  </Step>

  <Step title="Authenticate">
    Save, then complete the OAuth browser sign-in to your Cekura dashboard account — no API key needed.
  </Step>
</Steps>

<Note>
Custom connectors require a **Pro, Max, Team, or Enterprise** plan. On **Team / Enterprise**, an Owner must first add the connector under **Organization Settings**, after which members can enable it.
</Note>

## Add Cekura Skills (optional)

The connector gives Claude the tools; Skills teach it Cekura workflows (how to design evaluators, choose metrics, debug runs). Install the plugin from **Customize → Plugins → Personal plugins → + → Add marketplace → Add from a repository**, and enter `cekura-ai/cekura-skills`. The plugin also includes the MCP server, so it covers both. See the [Claude Desktop guide](/mcp/claude-desktop-guide) for the full plugin walkthrough.

## Verify setup

Start a new chat and ask:

```plaintext
Use the Cekura skills and MCP. List my Cekura agents, pick one, and propose 3 evaluators I should create first. Do not create anything until I approve.
```

Claude should list real agents from your workspace. With the connector only (no Skills), phrase requests with the workflow you want — metrics, evaluators, runs, call logs, or reports.

## Troubleshooting

<AccordionGroup>
  <Accordion title="Can't add a custom connector">
    Custom connectors need a Pro/Max/Team/Enterprise plan. On Team/Enterprise, ask an Owner to add the Cekura connector under **Organization Settings** first.
  </Accordion>

  <Accordion title="Connector won't connect">
    Claude reaches the server from Anthropic's cloud, so the MCP endpoint must be publicly reachable. Confirm the URL is exactly `https://api.cekura.ai/mcp`, then reconnect.
  </Accordion>

  <Accordion title="Authorization failed or wrong account">
    Reconnect the Cekura connector and confirm your browser is signed into the intended Cekura dashboard account.
  </Accordion>

  <Accordion title="I need API-key auth">
    OAuth is the recommended path. If your workspace requires API-key auth or custom headers, use the [API-key setup](/mcp/mcp-only#api-key-setup) or [Claude Code](/mcp/claude-code-guide).
  </Accordion>
</AccordionGroup>
