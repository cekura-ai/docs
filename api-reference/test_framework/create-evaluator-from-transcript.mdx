---
title: Create Evaluator from Transcript
description: "Create a test scenario from a call transcript"
openapi: post /test_framework/v1/scenarios/create_scenario_from_transcript/
slug: create-evaluator-from-transcript
---

<Note>
This endpoint creates a [Conditional Actions](/documentation/key-concepts/evaluators/conditional-actions) evaluator from the provided transcript. The response includes `scenario_type: "conditional_actions"` and the `instructions` field contains the generated conditional actions JSON.
</Note>
