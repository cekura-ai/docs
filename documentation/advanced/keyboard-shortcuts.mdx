Keyboard shortcuts let you navigate Cekura faster on Mac and Windows/Linux.

## Viewing Available Shortcuts

To view all shortcuts:

1. Click your profile icon in the top-right corner
2. Select **Keyboard Shortcuts** from the menu

You can also press `⌘/` on Mac or `Alt+/` on Windows/Linux.

## Available Shortcuts

### Navigation

| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| Open agent settings | ⌘E | Alt+E |
| Open settings | ⌘, | Alt+, |
| Create new agent | ⌘K | Alt+K |
| Show keyboard shortcuts | ⌘/ | Alt+/ |

### Actions

| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| Reload data | ⌘R | Alt+R |
| Select all items | ⌘A | Alt+A |
| Deselect all items | ⌘D | Alt+D |

<Note>
Some shortcuts are context-specific. For example, **Select All** and **Deselect All** only work on pages with selectable items like Evaluators, Results, Calls, and Metrics.
</Note>

<Tip>
Shortcuts are disabled when typing in input fields or text areas.
</Tip>
