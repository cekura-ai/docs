# Generate Metrics API

Automatically generate a set of evaluation metrics for an agent which can then be used to create metrics.

## API Endpoint

| Method | Endpoint |
|--------|----------|
| `POST` | `https://new-prod.vocera.ai/test_framework/v1/metrics-external/generate_metrics/` |

## Authentication

Include your API key in the request headers:

| Header | Description |
|--------|-------------|
| `X-VOCERA-API-KEY` | Your API key obtained from the dashboard |

## Request Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `agent_id` | integer | Yes | ID of the agent to generate metrics for |
| `num_metrics` | integer | Yes | Number of metrics to generate |

### Example Request Body

```json
{
    "agent_id": 1,
    "num_metrics": 5
}
```

## Response

The API returns an array of generated metric objects.

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `name` | string | Generated name for the metric |
| `description` | string | Detailed description of what the metric evaluates |
| `agent` | integer | ID of the associated agent |
| `eval_type` | string | Type of metric evaluation |

### Example Response

```json
[
    {
        "name": "introduction_clarity",
        "description": "This metric evaluates whether the AI voice agent clearly introduces itself and the purpose of the call...",
        "agent": 1,
        "eval_type": "binary_workflow_adherence"
    }
    // Additional metrics...
]
```

## Code Examples

<CodeGroup>

```bash Curl
curl -X POST https://new-prod.vocera.ai/test_framework/v1/metrics-external/generate_metrics/ \
  -H "X-VOCERA-API-KEY: <your-api-key-here>" \
  -H "Content-Type: application/json" \
  -d '{
    "agent_id": 1,
    "num_metrics": 5
  }'
```

```python Python
import requests

def generate_metrics(api_key, agent_id, num_metrics):
    url = 'https://new-prod.vocera.ai/test_framework/v1/metrics-external/generate_metrics/'
    
    headers = {
        'X-VOCERA-API-KEY': api_key,
        'Content-Type': 'application/json'
    }
    
    data = {
        'agent_id': agent_id,
        'num_metrics': num_metrics
    }

    response = requests.post(url, headers=headers, json=data)
    return response.json()
```

```javascript JavaScript
async function generateMetrics(apiKey, agentId, numMetrics) {
  const url = 'https://new-prod.vocera.ai/test_framework/v1/metrics-external/generate_metrics/';
  
  const response = await fetch(url, {
    method: 'POST',
    headers: {
      'X-VOCERA-API-KEY': apiKey,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      agent_id: agentId,
      num_metrics: numMetrics
    })
  });

  return await response.json();
}
```
