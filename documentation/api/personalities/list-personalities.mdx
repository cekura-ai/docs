# List Personalities API

Retrieve a list of personalities from your account

## API Endpoint

| Method | Endpoint                                                          |
| ------ | ----------------------------------------------------------------- |
| `GET`  | `https://api.cekura.ai/test_framework/v1/personalities-external/` |

## Authentication

Include your API key in the request headers:

| Header             | Description                              |
| ------------------ | ---------------------------------------- |
| `X-VOCERA-API-KEY` | Your API key obtained from the dashboard |

## Response Format

The API returns a list of personality objects.

### Personality Object

| Property      | Type    | Description                           |
| ------------- | ------- | ------------------------------------- |
| `id`          | integer | Unique identifier for the personality |
| `name`        | string  | Name of the personality               |
| `description` | string  | Description of the personality        |
| `language`    | string  | ISO 639-1 language code (e.g., "en")  |

## Example Response

```json
[
  {
    "id": 1,
    "name": "Highly Interruptive American Man",
    "description": "Highly Interruptive American Man",
    "language": "en"
  }
  // Additional personalities...
]
```

## Code Examples

<CodeGroup>

```bash Curl
curl -X GET https://api.cekura.ai/test_framework/v1/personalities-external/ \
  -H "X-VOCERA-API-KEY: <your-api-key-here>"
```

```python Python
import requests

def list_personalities(api_key):
    url = 'https://api.cekura.ai/test_framework/v1/personalities-external/'
    headers = {
        'X-VOCERA-API-KEY': api_key
    }

    response = requests.get(url, headers=headers)
    return response.json()
```

```javascript JavaScript
async function listPersonalities(apiKey) {
  const url = "https://api.cekura.ai/test_framework/v1/personalities-external/";

  const response = await fetch(url, {
    method: "GET",
    headers: {
      "X-VOCERA-API-KEY": apiKey,
    },
  });

  return await response.json();
}
```
