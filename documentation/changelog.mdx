## Recent Updates

<Update label="November 28, 2025" description="v2.5.0">
  # Project Metrics & Trend Based Alerts

  Powerful new features to streamline your AI agent testing and quality assurance workflow with project-level metrics, intelligent alerts, and enhanced testing capabilities.

  <Frame>
    <img src="https://mintlify-assets.b-cdn.net/hero-light.png" alt="Project Metrics Dashboard" />
  </Frame>

  ### Highlights

  - **Project Metrics** - Create metrics applicable across all agents, edited from one source with shared lab
  - **Trend Based Alerts** - Intelligent Slack/email alerts that detect anomalies based on metric patterns (Settings → Notifications → Observability Alerts)
  - **SMS Support During Call** - Test Agents can receive SMS between calls with full context for 2FA testing
  - **Test Profile in Expected Outcome** - Use `{{test_profile.variable_name}}` for templated evaluations
  - **New Integrations** - Agentforce chatbot and Kore AI voice agents support
</Update>

<Update label="November 15, 2025" description="v2.4.1">
  # Integrations, Real-World Simulations & Redaction

  Powerful new features to streamline your AI agent testing and quality assurance workflow with enhanced integrations, improved simulations, and data security.

  ### Highlights

  - **Pipecat WebRTC integration** - Automatically create rooms and connect to your agent via UI
  - **ElevenLabs WebSocket support** - Connect to agents via websockets for voice conversations
  - **Enhanced Real-World Simulations** - Create test cases from production calls (Observability → Actions → Create a Simulation)
  - **Redaction for Observability** - Enable redaction for sensitive data in transcripts and audio
  - **Automated Outbound Testing** - Test Retell/VAPI/Elevenlabs outbound voice agents without manual phone number setup
</Update>
