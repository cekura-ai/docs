## What You'll Learn

How to generate an API key, connect Cekura to your AI coding assistant (Claude Code, Cursor, VS Code), and use MCP to manage everything through natural language.

## Generate Your API Key

<Steps>
  <Step title="Navigate to Settings">
    Log in to [dashboard.cekura.ai](https://dashboard.cekura.ai) and go to **Settings → API Keys**.
  </Step>
  <Step title="Create a Key">
    Click **Create New Key** and copy the generated key. Store it securely — you'll need it for MCP, CI/CD, and API access.
  </Step>
</Steps>

<Warning>
Never commit API keys to version control. Use environment variables or secrets managers.
</Warning>

## Connect via MCP

MCP (Model Context Protocol) gives your AI assistant direct access to Cekura's documentation and APIs. One connection provides 84+ tools.

<Tabs>
  <Tab title="Claude Code">
    ```bash
    claude mcp add --transport http cekura --scope user \
      https://api.cekura.ai/mcp \
      --header "X-CEKURA-API-KEY:YOUR_API_KEY"
    ```

    Verify: `claude mcp list`
  </Tab>

  <Tab title="Cursor / VS Code">
    Add to your MCP config:

    ```json
    {
      "mcpServers": {
        "cekura": {
          "command": "npx",
          "args": [
            "mcp-remote",
            "https://api.cekura.ai/mcp",
            "--header",
            "X-CEKURA-API-KEY:${CEKURA_API_KEY}"
          ],
          "env": {
            "CEKURA_API_KEY": "YOUR_API_KEY"
          }
        }
      }
    }
    ```
  </Tab>
</Tabs>

### Try It

After connecting, ask your AI assistant:

```
List my Cekura agents
```

If configured correctly, it will call the Cekura API and return your agents.

## Using the API Directly

Every operation in Cekura is also available via REST API. All requests require the `X-CEKURA-API-KEY` header:

```bash
curl https://api.cekura.ai/test_framework/v1/aiagents/ \
  -H "X-CEKURA-API-KEY: YOUR_API_KEY"
```

See the full [API Reference](/api-reference) for all endpoints.
