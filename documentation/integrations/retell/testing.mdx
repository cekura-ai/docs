## Prerequisites
- A Cekura account
- Retell API credentials

## Setting Up Retell Integration

### 1. Accessing Organization Settings
1. Log in to your [Cekura Dashboard](https://dashboard.cekura.ai)
2. Choose the **Organization** you want to configure Retell for
3. Click on **Settings** button in the header

<img src="/images/org-button-click.webp" alt="Organization Settings Navigation" />

### 2. Configuring Retell Authentication
1. Locate the **Provider API Keys** section in sidebar
2. Enter your **Retell API key** in the designated field
3. Click on **Save Changes** button

<img src="/images/retell/retell-api-setup.png" alt="Retell API Configuration" />

## Testing Your Agent

### 1. Create a New Agent
1. Navigate to the **Agents** dropdown from the sidebar
2. Click on **Create an Agent** button
3. Fill in the required details in form
4. Click **Create** button

<img src="/images/create-agent.png" alt="Create Agent" />
<img src="/images/create-agent-modal.png" alt="Creating Agent" />

### 2. Select Retell as your provider

 1. After creating the agent, you will be redirected to the agent's settings page
 2. Choose **Retell** from the **Provider** dropdown menu
 3. Copy your agent id from Retell into the Assistant ID field
 4. Click on **Save Changes** button

<img src="/images/choosing-agent-provider.png" alt="Choosing Agent Provider" />
