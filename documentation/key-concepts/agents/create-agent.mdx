## Create Agent

<Steps>
  <Step title="Navigate to Agent Creation">
    Go to **Agent Settings** from the left navigation bar. If you are a new user, click **Create an Agent** to get started.
  </Step>

  <Step title="Add Agent Details">
    Fill in the basic information for your agent:

    - **Agent Name**: A descriptive name for your agent (e.g., "Customer Support Bot", "Scheduling Assistant")
    - **Language**: The primary communication language for your agent (e.g., English, Spanish, French)
  </Step>

  <Step title="Add Agent Description">
    The agent description is a comprehensive overview of your agent's purpose, behavior, and operational guidelines. It should encapsulate all the instructions, prompts, and rules you provide to your AI agent.

    <Info>Most customers paste their **agent's main system prompt** here.</Info>

    By providing a clear agent description, you enable Cekura to **generate automatic, accurate evaluators** for your agent.

    <img src="/images/description.png" alt="Agent description field" />
  </Step>
</Steps>

## Agent Description Formats

Cekura is flexible with how you provide your agent description. Choose the format that matches your setup:

<AccordionGroup>
  <Accordion title="Single Prompt Agent">
    For agents with a single system prompt, copy and paste the prompt directly into the agent description field.
  </Accordion>

  <Accordion title="Multi-State Agent">
    If your agent has multiple states or nodes, you can structure the description as JSON:

    ```json
    [
      {
        "node_name": "greeting",
        "node_description": "Welcome the caller and ask how you can help..."
      },
      {
        "node_name": "scheduling",
        "node_description": "Handle appointment scheduling requests..."
      }
    ]
    ```

    <Tip>
    Even if you copy-paste all your prompts as one large block, Cekura can process it. JSON is the recommended approach for multi-state agents.
    </Tip>
  </Accordion>

  <Accordion title="Retell Workflows">
    Export your agent details as JSON from Retell:

    1. Go to **Agents** in the Retell dashboard
    2. Select your agent
    3. Click the **Export** button (downloads a `.json` file)
    4. Copy the file contents and paste into the agent description field

    ![Retell Export](/images/retell/agent_description_workflow.png)
  </Accordion>

  <Accordion title="Vapi Workflows">
    Export your agent details as JSON from Vapi:

    1. Go to **Workflows** in the Vapi dashboard
    2. Select your agent
    3. Click the **Code** button (top right)
    4. Copy the content and paste into the agent description field

    ![Vapi Export](/images/vapi/agent_description_workflow.png)
  </Accordion>
</AccordionGroup>

<Note>You can follow the same export approach for other providers that support JSON export.</Note>
