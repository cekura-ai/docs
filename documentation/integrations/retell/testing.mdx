## Prerequisites

- A Cekura account
- A deployed Retell Assistant.
- Your Retell **API Key**.
- The Retell **Assistant ID** of the voice agent you wish to test.

## Setting Up Retell Integration

You must configure your Retell API Key once at the organization level to allow Cekura to interact with your Retell account.

### 1. Accessing Organization Settings

1. Log in to your [Cekura Dashboard](https://dashboard.cekura.ai).
2. Navigate to the **Settings** tab.(Typically on the top left).

![Organization Settings Navigation](/Cekura_home.png)

### 2. Configuring Retell Authentication

1. Locate the **Retell Integration** section.
2. Enter your Retell **API Key** in the designated field.
3. Click **Save**.

> **Note:** This key authenticates your Cekura organization to communicate with the Retell API. Once saved, you can connect multiple Retell Assistants to Cekura.

![Retell API Configuration](/retell_providerapi.png)

## Testing Your Agent

After configuring the API key, you can create an Agent in Cekura and link it to your specific Retell Assistant.

### 1. Create a New Agent (If not done already)

1. Navigate to the **Agents** section using the side navigation bar.
2. Click the **Add Agent** or **Create Agent** button.

![Createanagent Pn](/createanagent.png)

### 2. Select Retell as your provider when creating a new agent

1. Navigate to the **Agents** section.
2. Click the **Add Agent** button.
3. **Select Retell** as your provider from the dropdown.
4. Copy your **Retell Assistant ID** from the Retell Dashboard and paste it into the **Assistant ID** field in Cekura.
5. Click **Save Changes** to finalize the Cekura Agent.

![Retell API Configuration](/retell_agentsettings.png)
