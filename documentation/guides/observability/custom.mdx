## Configure Custom Observability

Follow these steps to set up custom observability for your voice agents:

<Steps>
<Step title="Obtain your API key">
Log in to your [Cekura dashboard](https://dashboard.cekura.ai). Navigate to the settings to retrieve your API key. You can find it in the API Keys section in the sidebar. This key is essential for authenticating your requests and accessing observability features.

<img src="/images/cekura/get-cekura-api-key-1.png" alt="Navigate to API Keys" />
</Step>

<Step title="Create an agent">
Navigate to the dashboard and create an agent. This agent will be responsible for collecting and sending data to the observability platform.

<img src="/images/create-agent.png" alt="Create Agent" />
</Step>

<Step title="Define metrics">
Go to [dashboard.cekura.ai](https://dashboard.cekura.ai) and select your agent to start creating metrics. These metrics will help you monitor and analyze the performance of your systems.

<img src="/images/create-metrics-1.png" alt="Create Metric" />
</Step>

<Step title="Start pushing logs">
Use our API to start pushing logs to the observability platform. This will allow you to track and analyze log data in real-time, providing deeper insights into your system's operations.
</Step>
</Steps>

For detailed implementation examples and API specifications, refer to our [API Introduction](/api-reference/observability/post-observabilityv1observe).
