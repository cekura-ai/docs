## What are Agents on Cekura?

Agents on Cekura are AI entities that simulate human or system behavior during tests. They’re used to interact with evaluators, perform conversations, execute tool calls, and help measure performance metrics like success rate, latency, and accuracy—making it easier to test and optimize conversational or automation workflows.

<Tip>
  Agents in Cekura is your Voice or Chat AI agent, which you are testing not the evaluators (testing agents)
</Tip>

## Types of Agents

<AccordionGroup>
  <Accordion title="Voice Agents" defaultOpen icon="microphone">
    **Voice Agents** talk and listen like real people during calls.
  </Accordion>
  <Accordion title="Chat Agents" defaultOpen icon="comment">
    **Chat Agents** handle text-based conversations to test chat workflows.
  </Accordion>
  <Accordion title="SMS Agents" defaultOpen icon="message">
    **SMS Agents** send and receive text messages to test SMS-based flows.
  </Accordion>
</AccordionGroup>
