# Update Agent API

Update an existing AI agent's configurations.

## API Endpoint

| Method | Endpoint |
|--------|----------|
| `PATCH` | `https://new-prod.vocera.ai/test_framework/v1/aiagents-external/{agent_id}` |

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `agent_id` | string | Yes | Unique identifier of the agent or associated assistant ID  |

## Authentication

Include your API key in the request headers:

| Header | Description |
|--------|-------------|
| `X-VOCERA-API-KEY` | Your API key obtained from the dashboard |

## Request Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `agent_name` | string | No | Name of the agent |
| `contact_number` | string | No | Phone number in E.164 format |
| `inbound` | boolean | No | If true, agent will call you |
| `description` | string | No | Description of the agent's purpose |
| `language` | string | No | ISO 639-1 language code (e.g., "en") |
| `assistant_id` | string | No | Vapi or Retell assistant ID |
| `websocket_url` | string | No | URL for LLM websocket |

### Example Request Body

```json
{
    "agent_name": "Agent B",
    "contact_number": "+11234567890",
    "inbound": true,
    "description": "hello",
    "language": "en",
    "assistant_id": "",
    "websocket_url": ""
}
```

## Response

A successful request returns the updated agent details.

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | integer | Unique identifier for the agent |
| `agent_name` | string | Name of the agent |
| `contact_number` | string | Phone number in E.164 format |
| `inbound` | boolean | If true, agent will call you |
| `description` | string | Description of the agent's purpose |
| `language` | string | ISO 639-1 language code (e.g., "en") |
| `assistant_id` | string | Vapi or Retell assistant ID |
| `websocket_url` | string | URL for LLM websocket |

```python Python
import requests

def update_agent(api_key, agent_id, agent_name=None, contact_number=None, inbound=None, 
                 description=None, language=None, assistant_id=None, websocket_url=None):
    url = f'https://new-prod.vocera.ai/test_framework/v1/aiagents-external/{agent_id}'
    
    headers = {
        'X-VOCERA-API-KEY': api_key,
        'Content-Type': 'application/json'
    }
    
    data = {k: v for k, v in {
        'agent_name': agent_name,
        'contact_number': contact_number,
        'inbound': inbound,
        'description': description,
        'language': language,
        'assistant_id': assistant_id,
        'websocket_url': websocket_url,
    }.items() if v is not None}

    response = requests.patch(url, headers=headers, json=data)
    return response.json()
```

```javascript JavaScript
async function updateAgent(apiKey, agentId, {
  agent_name = null,
  contact_number = null,
  inbound = null,
  description = null,
  language = null,
  assistant_id = null,
  websocket_url = null
} = {}) {
  const url = `https://new-prod.vocera.ai/test_framework/v1/aiagents-external/${agentId}`;
  
  const data = {
    ...(agent_name !== null && { agent_name }),
    ...(contact_number !== null && { contact_number }),
    ...(inbound !== null && { inbound }),
    ...(description !== null && { description }),
    ...(language !== null && { language }),
    ...(assistant_id !== null && { assistant_id }),
    ...(websocket_url !== null && { websocket_url })
  };

  const response = await fetch(url, {
    method: 'PATCH',
    headers: {
      'X-VOCERA-API-KEY': apiKey,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify(data)
  });

  return await response.json();
}
```
