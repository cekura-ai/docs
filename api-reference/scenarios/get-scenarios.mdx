# Get Scenarios API
Retrieve a list of scenarios for an agent

## API Endpoint

| Method | Endpoint |
|--------|----------|
| `GET` | `https://new-prod.vocera.ai/test_framework/v1/scenarios-external/` |

## Authentication

Include your API key in the request headers:

| Header | Description |
|--------|--------|
| `X-VOCERA-API-KEY` | Your API key obtained from the dashboard |

## Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `agent_id` | integer | Yes | Filter scenarios by agent ID |

## Response Format

The API returns a paginated list of scenarios.

### Response Properties

| Property | Type | Description |
|----------|------|-------------|
| `count` | integer | Total number of scenarios |
| `next` | string\|null | URL for the next page of results |
| `previous` | string\|null | URL for the previous page of results |
| `results` | array | List of scenario objects |

### Scenario Object

| Property | Type | Description |
|----------|------|-------------|
| `id` | integer | Unique identifier for the scenario |
| `name` | string | Name of the scenario |
| `personality_name` | string | Name of the associated personality |
| `phone_number` | string | Phone number associated with scenario |

## Code Examples

<CodeGroup>

```bash Curl
curl -X GET https://new-prod.vocera.ai/test_framework/v1/scenarios-external/ \
  -H "X-VOCERA-API-KEY: <your-api-key-here>" \
  -G \
  -d "agent_id=<agent-id-number>"
```

```python Python
import requests

def get_scenarios(api_key, agent_id):
    url = 'https://new-prod.vocera.ai/test_framework/v1/scenarios-external/'
    headers = {
        'X-VOCERA-API-KEY': api_key
    }
    params = {
        'agent_id': agent_id,
    }

    response = requests.get(url, headers=headers, params=params)
    return response.json()
```

```javascript JavaScript
async function getScenarios(apiKey, agentId) {
  const baseUrl = 'https://new-prod.vocera.ai/test_framework/v1/scenarios-external/';
  const params = new URLSearchParams({ agent_id: agentId });
  const url = `${baseUrl}?${params}`;

  const response = await fetch(url, {
    method: 'GET',
    headers: {
      'X-VOCERA-API-KEY': apiKey
    },
  });

  return await response.json();
}
```
