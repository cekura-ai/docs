## Prerequisites

To successfully integrate your VAPI agent, you will need the following:

- A **Cekura account** and access to your Cekura Dashboard.
- A **VAPI account** with a deployed Voice Assistant.
- Your **VAPI API Key** (for Cekura's Organization Settings).
- Your VAPI **Assistant ID** (for the Cekura Agent configuration).

## Setting Up VAPI Integration

Before you can create an Agent in Cekura, you must configure your VAPI API Key at the organization level.

### 1. Accessing Organization Settings

1. Log in to your [Cekura Dashboard](https://dashboard.cekura.ai)
3. Navigate to the **Settings** tab (Typically on the top left).

![Organization Settings Navigation](/Cekura_home.png)

### 2. Configuring VAPI Authentication

1. Locate the **Provider API Keys** or **VAPI Integration** section.
2. Enter your VAPI **Private API Key** in the designated field.
3. Click **Save Changes**.

> **Note:** The VAPI Private API Key authenticates your Cekura organization to communicate with VAPI's services. This only needs to be set up once.

![VAPI API Configuration](/vapi_providerapi.png)

## Testing Your Agent

Once the VAPI key is saved, you can create a new agent and link it to a specific VAPI Assistant.

### 1. Create a New Agent (If not done already)

1. Navigate to the **Agents** section using the side navigation bar.
2. Click the **Add Agent** or **Create Agent** button.

![VAPI API Configuration](/createanagent.png)

### 2. Configure Agent Settings

1. In the agent creation interface, name your new Cekura Agent.
2. In the **Provider** dropdown, select **VAPI**.
3. Copy your VAPI **Assistant ID** and paste it into the **Assistant ID** field in Cekura.
4. Click **Save** to create and configure the agent.

> **Tip:** Your VAPI Assistant ID can be found in your VAPI Dashboard under the specific Assistant's settings.

![image.png](/vapi_agentsettings.png)
