---
title: Create Evaluators from Call Logs
description: "Create scenarios from call logs in background using Celery tasks."
openapi: post /observability/v1/call-logs/create_scenarios/
---

<Note>
This endpoint creates [Conditional Actions](/documentation/key-concepts/evaluators/conditional-actions) evaluators from call log transcripts. Each created scenario has `scenario_type: "conditional_actions"` and the `instructions` field contains the generated conditional actions JSON.
</Note>
